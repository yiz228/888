## PLACES County Health Data Project
This project uses CDC PLACES county-level data to examine geographic variation in arthritis prevalence across U.S. states.

## Research Question
In 2023, how does the prevalence of arthritis among adults vary across U.S. states?

## Background
Arthritis is a common chronic condition among U.S. adults and contributes to pain, disability, and reduced quality of life. This project explores whether county-level crude arthritis prevalence differs across states.

## Data Source
The data come from the CDC PLACES county-level dataset:
PLACES__Local_Data_for_Better_Health,_County_Data,_2025_release_20260127.csv

For this project, the analytic dataset focuses on:
- Year: 2023
- Measure: Arthritis among adults
- Data value type: Crude prevalence
